Pod search 找不到库的解决方案

Share

报错如下

删除本地目录后修复

使用cocoapods的时候，当cocoapods更新到新的版本的时候，pod install 会报此cocoapods没有满足的版本，要求更新cocoapods，应该是本地缓存的问题，在这里记录一下解决办法，方便以后查找。

  • 移除本地master
  • 移除本地缓存
  • 重新setup,如果网速较慢,可以在后面加上 –verbose
